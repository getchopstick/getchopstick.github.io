Get Started

Chopstick can be used as a styleguide, to build static websites or to speed up making HTML prototypes.

TL;DR

  • Step 1
    git clone https://github.com/getchopstick/chopstick-boilerplate.git
  • Step 2
    sh install.sh
  • Step 3
    gulp watch

Chopstick makes use of the Chopstick Sass framework. It adds tools like Jekyll and Gulp to simplify designing and building in the browser.
