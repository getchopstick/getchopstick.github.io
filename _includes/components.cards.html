Design framework

Chopstick is a mobile-first responsive framework written in Sass. It uses the ITCSS architecture and BEMit naming methodology. It enables you to quickly create custom design systems.

Get it on Github

Framework documentation

Requirements

Bower & Sass

TL;DR

  • Step 1
    git clone https://github.com/getchopstick/chopstick.git
  • Step 2
    bower install

Boilerplate

The Chopstick boilerplate is a prototyping tool built around Jekyll, Gulp and the Chopstick framework. It enables you to quickly create high-fidelity prototypes, marketing sites and blogs.

Get it on Github

Boilerplate documentation

Requirements

Bower, Gulp & Jekyll

TL;DR

  • Step 1
    git clone https://github.com/getchopstick/chopstick-boilerplate.git
  • Step 2
    bundle install && npm install && bower install
